# Review Bundles

이 폴더는 사용자가 별도로 요청했던 검토용 파일 묶음을 GitHub에서 확인하기 쉽도록 복사한 것이다.

## 포함 폴더

- `full_supervised_matrix_v1_requested_files/`: locked full supervised matrix의 subject/class/confusion metrics
- `literature_full_extension_v1_requested_files/`: literature extension 결과 검토용 핵심 파일
- `v3_attention_token_importance_inference_only/`: inference-only v3 attention/token importance 산출물

## 주의

이 파일들은 원본 결과에서 선별 복사한 검토용 bundle이다. 원본 result directory는 수정하지 않았다.
